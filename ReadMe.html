Form Login by Riyan Satria

Cara Pasang

  1. Buka file konfig.php dan ganti dengan informasi database Anda
  2. Buat Database baru dengan nama Login
  3. Buka PhpMyAdmin DB Anda, klik Bagian Import
  4. Pilih File login.sql di dalam folder inc dan klik tombol Go
  5. Upload semua file pada direktori terserah (yang penting di dalam direktori public_html)
